How to Edit Content

Change Text Content

Open any .html file and edit the text directly the HTML tags.

Change Images

All images are located in the images/ folder. Replace an image with your own while keeping the same file name, or update the image path in HTML.

Change Icons

Icons are located in the icons/ folder or loaded via external icon libraries. Official source links can be found directly in the code.

Customization

Main colors can be changed in css/style.css. Search for:

:root or color variables at the top of the file. 
:root {
  --primary: #c7a17a;
  --secondary: #f3ebe4;
}

Favicon

You can replace the favicon files located in the root folder with your own branding icons.

JavaScript

All JavaScript functionality is located in js/, including:

  • Interactive services
  • Carousel/sliders
  • Booking form logic

Each section in the JavaScript file is commented for easier understanding and customization
