# SPA Landing Page – Premium Website Template

A modern, responsive SPA & Massage Salon landing page template designed for commercial use.
Check out the template preview here: **[SPA Template Preview](https://ghostlayer168.github.io/spa-html-template/preview/)**
See the template in action: **[SPA Template Demo](https://ghostlayer168.github.io/spa-html-template/demo/)**

## Overview

SPA Landing Page is a professionally designed website template built for:

- Spa salons
- Massage studios
- Wellness centers
- Beauty businesses

The template features a clean UI, smooth layout, and responsive design optimized for desktop, tablet, and mobile devices.

## Features

- Fully responsive layout
- Modern and clean design
- Optimized structure
- Multi-page layout
- Booking page included
- Service pages included
- Lightweight and fast loading

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- WebP optimized images
